## Project Overview


During my PhD, I explored several topics before finally focusing on the security of in-vehicle communication,
 especially the Controller Area Network (CAN) bus. CAN bus is widely used in vehicles 
 to allow different parts like brakes, engines, and sensors to communicate with each other.
 However, it was not designed with security in mind, making it vulnerable to cyber attacks. My PhD journey focused on 
 how to secure CAN Bus.


## Key Outputs

- **Evaluation of CAN Bus Security Challenges**  
  A highly cited and comprehensive review paper that summarizes the main security issues in the CAN bus protocol and existing solutions.

- **WINDS: A Wavelet-Based Intrusion Detection System for Controller Area Network (CAN)**  
  Proposes a novel IDS that uses wavelet transforms to detect unusual patterns in CAN signals.

- **Hardware Trojan Enabled Denial of Service Attack on CAN Bus**  
  Demonstrates a practical attack on the CAN bus using a hardware trojan to launch a Denial of Service (DoS).

- **Temporal Logic-Based Intrusion Detection for Securing Connected Vehicles**  
  Introduces an IDS based on signal timing and logic rules, designed to detect attacks with low resource usage in embedded systems.
